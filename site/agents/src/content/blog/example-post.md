## Features

- **Durable Execution**: Build agents that can handle long-running tasks
- **Serverless Inference**: Leverage Cloudflare's edge network for fast responses
- **Scalable Pricing**: Pricing that scales up and down with your usage

## Getting Started

To get started with Cloudflare Agents, simply run:

```bash
npm i agents
```

Then follow our [documentation](https://developers.cloudflare.com/agents) to build your first agent.
